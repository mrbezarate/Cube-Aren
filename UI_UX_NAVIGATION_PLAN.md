# 📋 UI/UX Navigation Improvement Plan v2
## Underground Arena - Современная Архитектура Навигации

**Дата:** 2 Июля 2026  
**Статус:** Планирование (не реализовано)  
**Цель:** Создать интуитивную многоуровневую навигацию с акцентом на мобильный UX

---

## 💡 Новая Концепция

**Принцип:** Не перегружать navbar, использовать современные паттерны навигации

1. **Navbar (Топ-бар)** - только главные разделы (3-4 пункта максимум)
2. **Sidebar (Боковая панель)** - для авторизованных пользователей (социальные функции)
3. **Bottom Tab Bar (Мобильная панель)** - для мобильных устройств
4. **FAB (Floating Action Button)** - для быстрых действий
5. **Profile Hub** - централизованный личный кабинет вместо разбросанных страниц

## ❌ Что Убираем и Почему

### **1. "Кабинет" (`/dashboard`) - УДАЛИТЬ ❌**

**Почему странно:**
- Непонятное название ("кабинет" - что это?)
- 4 разных вкладки в одном месте (wallet, bets, tournaments, saved)
- Нелогично - почему мои ставки и кошелёк в одном месте?
- Устаревший подход (как в старых админках)

**Что вместо:**
- **Profile Hub** (`/profile/[id]`) - твоя главная страница
- Вкладки в профиле: Обзор | Турниры | Команды | Ставки | Достижения
- Отдельная страница **Wallet** (`/wallet`) для финансов
- Отдельная страница **Saved** (`/saved`) для сохранённого

---

### **2. Navbar перегружен - УПРОСТИТЬ ⚠️**

**Текущая проблема:**
- Слишком много пунктов для авторизованных (Турниры, Сообщество, Создать, Кабинет)
- Не масштабируется (если добавим ещё функции - некуда ставить)
- Мобильная версия страдает (hamburger menu с кучей пунктов)

**Решение:**
- Navbar: только 3 основных раздела (Турниры, Сообщество, Рейтинг)
- Sidebar: личные разделы (Чат, Друзья, Команды, Сохранённое)
- Dropdown: профиль, настройки, кошелёк

**Функционал:**
1. **Посты от команд/организаторов:**
   - Официальные команды могут постить новости
   - Организаторы могут анонсировать турниры
   - Администраторы могут постить обновления платформы

2. **Комментарии:**
   - Любой авторизованный может комментировать
   - Вложенные ответы (до 3 уровней)
   - Лайки на комментарии

3. **Реакции:**
   - Лайки (❤️)
   - Просмотры (👁️)
   - Сохранить в избранное (💾)

4. **Фильтры:**
   - По играм (CS2, Dota 2, Valorant и т.д.)
   - По типу (Новости, Обсуждения, Анонсы)
   - По популярности / по времени

5. **Уведомления:**
   - Новый пост от команды, на которую подписан
   - Ответ на твой комментарий
   - Лайк на твой пост/комментарий

**Backend:**
- Entity: `Post` (title, content, authorId, gameType, type, createdAt)
- Entity: `Comment` (postId, userId, content, parentId, createdAt)
- Entity: `PostReaction` (postId, userId, type: 'like')
- API endpoints: CRUD для постов, комментариев, реакций

**Кто может постить:**
- ✅ Организаторы (organizer role)
- ✅ Администраторы (admin role)
- ✅ Верифицированные команды (будущая фича)
- ❌ Обычные пользователи (пока не могут создавать посты, только комментировать)

### **Вкладки в профиле:**

**1. Обзор (Overview)** - текущая страница профиля
- Статистика (wins, losses, winrate, arena power)
- Последние турниры
- Команды
- Любимые игры

**2. Турниры (Tournaments)** - из старого dashboard
- История участия в турнирах
- Результаты (места, призы)
- Статистика по играм
- Фильтры: По игре, По дате, По результату

**3. Команды (Teams)**
- Список всех команд пользователя
- Роль в команде (Captain, Member)
- Статистика по командам
- Заявки в команды (если капитан)

**4. Ставки (Bets)** - из старого dashboard
- История ставок
- Выигрыши/проигрыши
- Статистика по ставкам
- Фильтры: Активные, Завершённые, Выигранные, Проигранные

**5. Достижения (Achievements)** - будущая фича
- Badges (иконки за достижения)
- Трофеи (за победы в турнирах)
- Прогресс (уровни, опыт)

### **Логика:**
- Для **своего профиля** (`/profile/[myId]`) - показываем все вкладки + кнопка "⚙️ Настройки"
- Для **чужого профиля** (`/profile/[otherId]`) - показываем публичные вкладки (Обзор, Турниры, Команды)
- **Кнопка "⚙️ Settings"** в правом верхнем углу профиля (только для своего)

## 📱 Мобильная Навигация (Детальный План)

### **Проблема с navbar на мобильных:**
- Слишком много пунктов не влезают
- Hamburger menu - неудобно (нужно 2 клика)
- Текст занимает много места

### **Решение: Гибридный подход**

#### **Desktop (>1024px):**
```
Navbar (топ):     [Logo] Турниры | Сообщество | Рейтинг   [💰] [🔔] [👤]
Sidebar (слева):  Главная, Чат, Друзья, Команды, Сохранённое, Настройки
```

#### **Tablet (768px - 1024px):**
```
Navbar (топ):     [Logo] Турниры | Сообщество | Рейтинг   [💰] [🔔] [👤]
Sidebar (слева):  Иконки (сворачивается в узкую полоску)
```

#### **Mobile (<768px):**
```
Navbar (топ):     [Logo] Underground                             [🔔] [☰]
Bottom Tab Bar:   🏠  🏆  💬  👥  👤
                 Дом Турн Чат Друз Проф
```

**Логика:**
- На мобильных **убираем текстовые пункты** из navbar
- Navbar только: Logo + Уведомления + Меню
- Всю навигацию переносим в **Bottom Tab Bar**
- 5 вкладок: Главная, Турниры, Чат (или Сообщество для неавт.), Друзья, Профиль

### **Bottom Tab Bar - 2 варианта**

**Для НЕ авторизованных:**
```
┌──────────────────────────────────────────────────────────┐
│   🏠         🏆         📰         📊         👤          │
```

- Чат: количество непрочитанных сообщений
- Друзья: количество новых запросов на дружбу
