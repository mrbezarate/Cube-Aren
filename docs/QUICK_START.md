# 🚀 Быстрый старт - Новая система друзей

## ✅ Что изменилось

### Было (старая система):
- ❌ Запросы в друзья (friend requests)
- ❌ Таблица friendships
- ❌ Кнопки "Добавить в друзья", "Принять запрос"
- ❌ Сложная логика с 3 таблицами

### Стало (TikTok style):
- ✅ Только подписки (follows)
- ✅ Друзья = взаимные подписки
- ✅ Кнопка "Подписаться" / "Отписаться"
- ✅ Простая логика с 1 таблицей

## 🎮 Как работает

### Шаг 1: Подписка
```
User A → нажимает "Подписаться" на User B
```
- Создаётся Follow (A → B)
- User B получает +1 подписчик
- User A получает +1 подписка

### Шаг 2: Входящие
```
User B → заходит в "Входящие"
```
- Видит User A в списке
- User A показывается потому что подписался, но User B ещё не подписался в ответ

### Шаг 3: Взаимная подписка (Друзья)
```
User B → нажимает "Подписаться в ответ" на User A
```
- Создаётся Follow (B → A)
- ТЕПЕРЬ ОНИ ДРУЗЬЯ (взаимная подписка)
- Оба видят друг друга во вкладке "Друзья"
- Появляется кнопка "Написать" (чат)

### Шаг 4: Чат
```
User A или User B → нажимает "Написать"
```
- Открывается чат
- Можно отправлять сообщения
- Чат работает ТОЛЬКО между друзьями (взаимные подписки)

## 🗂️ Вкладки на странице /friends

### 1. Друзья
- Показывает всех с кем взаимная подписка
- Кнопка "Написать" (открыть чат)
- Кнопка "Отписаться"

### 2. Входящие
- Показывает тех, кто подписался на тебя
- НО ты ещё не подписался на них
- Кнопка "Подписаться в ответ"

### 3. Найти друзей
- Поиск по нику
- Показывает статусы:
  - 🟢 "Друзья" - взаимная подписка
  - 🔵 "Вы подписаны" - ты подписан, он нет
  - 🟣 "Подписан на вас" - он подписан, ты нет
- Кнопки в зависимости от статуса

## 📱 Профиль пользователя

### Если ты НЕ подписан:
- Кнопка "Подписаться"

### Если он подписан на тебя:
- Кнопка "Подписаться в ответ"

### Если ты уже подписан:
- Кнопка "Отписаться"

### Если вы друзья (взаимная подписка):
- Кнопка "Написать" (чат)
- Кнопка "Отписаться"

## 🔢 Счётчики

- **followersCount** - количество подписчиков (кто подписан на тебя)
- **followingCount** - количество подписок (на кого ты подписан)
- Друзья = пересечение этих списков (взаимные)

## 🎯 Примеры

### Сценарий 1: Хочу добавить друга
```
1. Иду в "Найти друзей"
2. Ищу по нику "test2"
3. Нажимаю "Подписаться"
4. Жду пока он подпишется в ответ
5. Когда он подпишется - становимся друзьями
6. Появляется в "Друзья", могу написать
```

### Сценарий 2: Кто-то подписался на меня
```
1. Захожу во "Входящие"
2. Вижу кто подписался
3. Если хочу дружить - нажимаю "Подписаться в ответ"
4. Становимся друзьями
5. Появляется кнопка "Написать"
```

### Сценарий 3: Хочу написать другу
```
1. Открываю вкладку "Друзья"
2. Выбираю друга
3. Нажимаю кнопку с иконкой сообщения
4. Открывается чат
5. Пишу сообщение
```

## 📊 Статусы в поиске

| Бейдж | Значение | Кнопка |
|-------|----------|--------|
| 🟢 Друзья | Взаимная подписка | Отписаться |
| 🔵 Вы подписаны | Ты подписан, он нет | Отписаться |
| 🟣 Подписан на вас | Он подписан, ты нет | Подписаться в ответ |
| (нет бейджа) | Нет подписок | Подписаться |

## ✨ Преимущества

1. **Проще**: Понятная логика как в TikTok
2. **Быстрее**: Меньше кликов для дружбы
3. **Гибче**: Можно подписываться без "запросов"
4. **Знакомо**: Все знают как работает подписка

## 🐛 Если что-то не работает

1. **Не вижу входящие**:
   - Убедитесь что кто-то подписался на вас
   - Проверьте что вы НЕ подписаны на них

2. **Не вижу друзей**:
   - Друзья = взаимная подписка
   - Оба должны быть подписаны друг на друга

3. **Чат не работает**:
   - Чат работает ТОЛЬКО между друзьями
   - Проверьте что есть взаимная подписка

4. **Счётчики не обновляются**:
   - Обновите страницу
   - Счётчики обновляются при подписке/отписке

Всё работает! 🎉
