# 🎯 Система настроек Underground Arena

Полноценная система настроек с 5 вкладками для управления профилем и предпочтениями пользователя.

## 📁 Структура маршрутов

```
/settings/
├── profile        - Настройки профиля (редактирование данных)
├── privacy        - Конфиденциальность
├── account        - Аккаунт и безопасность
├── notifications  - Уведомления
└── preferences    - Предпочтения
```

**Редирект:** `/profile/edit` → `/settings/profile`

## 🎨 Frontend реализация

### Компоненты

1. **Layout** (`/settings/layout.tsx`)
   - Общий layout со боковой навигацией
   - Sticky sidebar с активной подсветкой
   - Кнопка возврата к профилю

2. **Страницы настроек:**
   - ✅ `/settings/profile` - Редактирование профиля
   - ✅ `/settings/privacy` - Настройки конфиденциальности
   - ✅ `/settings/account` - Управление аккаунтом
   - ✅ `/settings/notifications` - Уведомления
   - ✅ `/settings/preferences` - Предпочтения

### Типы (TypeScript)

Добавлены в `frontend/src/types/index.ts`:

- `PrivacyLevel` - уровни приватности
- `ProfileVisibility` - видимость профиля
- `PrivacySettings` - настройки конфиденциальности
- `NotificationSettings` - настройки уведомлений
- `UserPreferences` - пользовательские предпочтения
- `BlockedUser` - заблокированный пользователь

## 🔧 Backend реализация

### Entities

Созданы новые entity файлы в `backend/src/entities/`:

1. **privacy-settings.entity.ts**
   - Видимость профиля (public/friends/private)
   - Кто может писать, видеть статистику, друзей, приглашать
   - Показывать онлайн статус
   - Показывать историю посещений и турниров

2. **notification-settings.entity.ts**
   - Email уведомления (8 типов)
   - Push уведомления (5 типов)
   - In-app уведомления (3 типа)

3. **user-preferences.entity.ts**
   - Язык (ru/en/ua)
   - Тема (dark/light/system)
   - Цветовой акцент (purple/blue/green/gold)
   - Часовой пояс, форматы даты/времени
   - Фильтры турниров
   - Настройки производительности
   - Контент-фильтры

4. **blocked-user.entity.ts**
   - Связь пользователь-заблокированный
   - Причина блокировки
   - Дата блокировки

### База данных

**Миграция:** `003_add_user_settings_tables.sql`

Создаёт 4 таблицы:
- `privacy_settings`
- `notification_settings`
- `user_preferences`
- `blocked_users`

С индексами, триггерами и дефолтными значениями для существующих пользователей.

## 📋 Функциональность по вкладкам

### 1️⃣ Профиль (`/settings/profile`)

**Что можно настроить:**
- ✅ Имя пользователя (username) + cooldown 3 дня
- ✅ Отображаемое имя (displayName)
- ✅ Короткий статус (tagline)
- ✅ О себе (bio) - до 500 символов
- ✅ Аватарка + превью + cooldown 7 дней
- ✅ Баннер + превью + cooldown 7 дней
- ✅ Пол + cooldown 7 дней
- ✅ Страна/город
- ✅ Основная игра
- ✅ Любимые игры (до 5)

**Особенности:**
- Cooldown индикаторы с предупреждениями
- Предпросмотр изображений
- Счётчик символов для bio

### 2️⃣ Конфиденциальность (`/settings/privacy`)

**Видимость профиля:**
- 🌐 Публичный (все видят)
- 👥 Только друзья
- 🔒 Приватный (никто не видит)

**Кто может:**
- Писать в чат
- Видеть статистику
- Видеть друзей
- Приглашать в команду
- Видеть онлайн статус

**История:**
- ✅ Показывать историю посещений (только мне)
- ✅ Показывать историю турниров
- 🗑️ Очистить историю посещений
- 🗑️ Очистить историю турниров

**Блокировка:**
- 🚫 Список заблокированных пользователей
- Заблокированные не могут: писать, видеть профиль, приглашать

### 3️⃣ Аккаунт (`/settings/account`)

**Email:**
- Текущий email + статус подтверждения
- Изменение email (с подтверждением)

**Пароль:**
- Изменение пароля (текущий + новый + повтор)
- Только для local auth

**2FA:**
- 🔐 Включить/отключить двухфакторную аутентификацию
- Поддержка Google Authenticator / Email

**Связанные аккаунты:**
- 🔗 Google: подключить/отключить
- 🎮 Discord: подключить/отключить

**Опасная зона:**
- 📥 Экспорт данных (GDPR compliance)
- 🗑️ Удалить аккаунт (с задержкой 30 дней)

### 4️⃣ Уведомления (`/settings/notifications`)

**Email (8 типов):**
- ✅ Новый турнир в любимых играх
- ✅ Турнир скоро начнётся (за 1 час)
- ✅ Результаты ставки
- ✅ Новая заявка в команду
- ✅ Приглашение в команду
- ✅ Новое сообщение (если не читал 24ч)
- ✅ Еженедельная рассылка
- ❌ Маркетинговые предложения (по умолчанию выключено)

**Push (5 типов):**
- ✅ Новое сообщение в чате
- ✅ Кто-то подписался на вас
- ✅ Турнир начался
- ✅ Ваша ставка выиграла/проиграла
- ✅ Новая заявка в команду (для капитана)

**В приложении (3 типа):**
- ✅ Показывать badge на новых сообщениях
- ✅ Показывать badge на входящих запросах
- ✅ Уведомления в навбаре (колокольчик)

### 5️⃣ Предпочтения (`/settings/preferences`)

**Интерфейс:**
- 🇷🇺 Язык: Русский / English / Українська
- 🌙 Тема: Тёмная / Светлая / Системная
- 🎨 Цветовой акцент: Фиолетовый / Синий / Зелёный / Золотой

**Отображение:**
- ⏰ Часовой пояс: UTC+3 (Москва) / Автоопределение
- 📅 Формат даты: DD.MM.YYYY / MM/DD/YYYY / YYYY-MM-DD
- 🕐 Формат времени: 24-часовой / 12-часовой (AM/PM)

**Игры:**
- ✅ Скрывать неинтересные турниры
- ✅ Показывать только региональные турниры
- 💰 Фильтр по минимальному призовому фонду: 0 / 1k / 5k / 10k

**Производительность:**
- ✅ Включить анимации
- ✅ Автовоспроизведение видео
- ✅ Предзагрузка изображений
- 🖼️ Качество изображений: Высокое / Среднее / Низкое

**Контент:**
- ❌ Показывать контент для взрослых (18+)
- ❌ Фильтровать нецензурную лексику в чате
- ✅ Скрывать спойлеры автоматически
